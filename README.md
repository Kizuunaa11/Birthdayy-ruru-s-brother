# Birthday Cinematic Experience

A modern interactive birthday web experience built with Next.js App Router, React, Tailwind CSS, Framer Motion, GSAP, React Three Fiber, and Howler.js.

## Features
- Cinematic intro with blur-to-focus reveal
- Smooth scene transitions
- Interactive 3D gift box reveal
- Cake reveal scene with warm lighting and confetti
- Subtle sound effects and floating particle atmosphere

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```

## Notes
- The project requires internet access for npm package install and sound asset loading.
- If `npm install` fails due to network or permission issues, retry with a stable connection.
